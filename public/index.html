🇩🇪 DW German Learning

Learn German with Deutsche Welle's slow news

📅 Status: The application is ready to run! Content will be generated when you execute the application or when the daily automation runs.

Features:

📰 Daily German News: Fetches content from Deutsche Welle's "Langsam gesprochene Nachrichten"
🤖 AI Explanations: Explains difficult German words with definitions and examples
❓ Interactive Quizzes: Tests your comprehension and vocabulary
🎧 Audio Support: Listen to pronunciation and practice
📱 Responsive Design: Works perfectly on all devices
🚀 Getting Started:
1. Configure your GitHub token in the environment variables
2. Run npm run generate to create your first content
3. The website will be generated in the public/ directory
